# Lempel-Ziv (book) – Instructions

1. Find a pdf file of your favorite book and read the contents of the file into a byte array.
2. Change bytes to bits to obtain a representation of the pdf file of the book as a string of bits.
3. Using a starting dictionary of 256 bits use Lempel-Ziv algorithm to compress the obtained string of bits.
4. Compare the size of the compressed file with the sizes of the same book in some other document formats (DjVu etc.).
